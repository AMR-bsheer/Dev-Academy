محتوى المسار

في مسار تطوير التطبيقات الخلفية، ستتعرف على أفضل الممارسات وأدوات البرمجة لتطوير تطبيقات ويب قابلة للتوسع وقابلة للصيانة. ستتعلم كيفية إنشاء خدمات ويب، التعامل مع قواعد البيانات، وتأمين التطبيقات

code

لغة البرمجة php

اكتسب مهارات برمجة الويب باستخدام لغة PHP الشهيرة.

framework

إطار العمل Laravel

اكتشف كيفية تطوير تطبيقات ويب قوية باستخدام إطار Laravel.

database

التعامل مع قواعد البيانات MySQL

تعلم إدارة قواعد البيانات وتخزين البيانات باستخدام MySQL.

api

دارة API باستخدام RESTful

تعلم كيفية بناء واجهات برمجية تفاعلية باستخدام RESTful API.
